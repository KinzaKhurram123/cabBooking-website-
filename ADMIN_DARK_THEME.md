## 🎨 Dark Theme Features

### 1. **Color Scheme**
- **Background**: Deep dark (#0f1419)

- **Accents**: Purple gradient (#8b5cf6)

### 2. **Sidebar**
- Dark gradient background (#1a1d29 to #2d1b3d)
- Purple accent border
- Hover effects with purple glow
- Active menu items with purple highlight

### 3. **Top Bar**
- Dark background with purple border
- Light text colors
- Notification icon with purple hover

### 4. **Stats Cards**
- Dark background (#1a1d29)
- Purple border glow
- Hover effect with purple shadow
- Gradient icons with glow effects:
  - Blue/Purple gradient
  - Green gradient
  - Orange gradient
  - Pink gradient

### 5. **Charts**
- Dark card backgrounds
- Purple chart lines (#8b5cf6)
- Dark grid lines with purple tint
- Light gray axis labels
- Updated doughnut chart colors:
  - Completed: Green (#10b981)
  - In Progress: Purple (#8b5cf6)
  - Cancelled: Red (#ef4444)
  - Pending: Orange (#f59e0b)

### 6. **Recent Activity**
- Dark background
- Purple border accents
- Light text colors
- Subtle dividers
