## ✨ Implemented Features

### 🎨 Design & Animations
- **Animated gradient background** with smooth color transitions

- **Hover effects** on all interactive elements

- **Google Places Autocomplete** for both pickup and destination

- **Real-time route calculation** from pickup to drop-off

### 🚗 Service Selection
- **Cab Ride** - Comfortable transportation
- **Parcel Delivery** - Quick package delivery

- **Visual service cards** with hover animations
- **Active state indication** for selected service

### 💰 Ride Details
- **Distance calculation** in km/miles

- **CSS animations and transitions**

## 🎮 How to Use

1. **Open Dashboard**: Navigate to `dashboard.html` or click "Dashboard" in the main menu
2. **Enter Locations**: 
   - Type your pickup location in the first search box
   - Type your destination in the second search box
   - Select from autocomplete suggestions
3. **Select Service**: Click on Cab Ride, Parcel Delivery, or Pet Delivery
4. **View Route**: The map automatically shows the route with distance and fare
5. **Book Ride**: Click "Book Now" to confirm your booking
